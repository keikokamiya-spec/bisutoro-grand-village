# WordPress 管理対象

このフォルダの `wp-theme/grand-village/` が WordPress 用テーマです。
WordPress の `wp-content/themes/grand-village/` に配置して有効化してください。

## 1. ブログ

通常の「投稿」で管理します。

- お知らせ: カテゴリースラッグ `information`
- ランチ: カテゴリースラッグ `lunch`
- 黒板メニュー: カテゴリースラッグ `kokuban`

単記事ページの木目見出しはカテゴリに応じて表示されます。
ランチカテゴリの記事では `お知らせ` ではなく `ランチ` と表示されます。

## 2. 写真

管理画面に専用投稿タイプを追加しています。

- `ギャラリー写真`: ギャラリーページに表示
- `黒板メニュー写真`: 黒板メニューページに表示

各投稿で「アイキャッチ画像」を設定してください。
並び順は「順序」または投稿日時で調整できます。

## 3. 一部テキスト（料金）

管理画面の `外観 > カスタマイズ > 料金設定` で編集できます。
現在はお子様セットメニューの料金を管理対象にしています。

- ベーコンとキノコのトマトスパゲッティ
- みんな大好き カルボナーラ
- いろいろキノコのクリームリゾット
- 煮込みハンバーグのデミグラスソース

対象を増やす場合は `functions.php` の `gv_price_defaults()` に項目を追加し、該当テンプレートで `gv_price('キー名')` を使います。
