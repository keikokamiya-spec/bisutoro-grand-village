# Bistrot Grand Village WordPress Theme Upload Lite

WordPress 管理画面の「テーマアップロード」で入れやすくするための軽量版テーマです。

## アップロード

1. この `bistro-grand-village-theme-local-priority-admin-upload-lite` フォルダを zip 化します。
2. WordPress 管理画面で「外観 > テーマ > 新規追加 > テーマのアップロード」から zip をアップロードします。
3. テーマを有効化します。
4. ACF 無料版プラグインを有効化します。

## 軽量版の注意

- ホームの基本デザイン、ヘッダー、フッター、コンセプト、シェフ紹介、アクセス、内観外観は同梱されています。
- 容量を下げるため、重いギャラリー画像、料理リンク用画像、黒板メニューのデフォルト画像は同梱していません。
- それらは有効化後に ACF から登録してください。

## 管理できる内容

- ランチメニュー: 管理画面の「ランチメニュー」から追加・編集します。
- お子様セット: 管理画面の「お子様セット」から追加・編集します。
- ドリンクメニュー: 管理画面の「ドリンクメニュー」から追加・編集します。
- ワインリスト: 管理画面の「ワインリスト」から追加・編集します。
- 料理画像リンク: 管理画面の「料理画像リンク」から追加・編集します。
- 黒板メニュー: 固定ページ「黒板メニュー」の編集画面で見出し、説明文、画像を変更します。
- ギャラリー: 固定ページ「ギャラリー」の編集画面で画像を変更します。
- 営業時間＆アクセス: 固定ページ「営業時間＆アクセス」の編集画面で店舗情報、営業時間、地図URLを変更します。

## 固定ページ

テーマ有効化時に以下の固定ページを作成し、テンプレートを割り当てます。

- トップページ
- 黒板メニュー
- ランチ
- ディナーメニュー
- お子様セット
- ドリンク
- ワインリスト
- 内観＆外観
- ギャラリー
- 営業時間＆アクセス

メニュー投稿が未登録の場合は、既存HTMLから移植した初期表示を使います。
