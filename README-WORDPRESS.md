# WordPress対応メモ

このディレクトリをWordPressのテーマとして `wp-content/themes/` に配置し、管理画面からテーマを有効化してください。

## 編集できるページ

テーマ有効化時に、以下の固定ページが作成されます。

- 黒板メニュー
- ランチ
- ディナーメニュー
- お子様セット
- ドリンク
- ワインリスト
- 内観＆外観

これらはWordPress管理画面の「固定ページ」から本文を編集できます。本文エリアは左寄せで表示されるようにしています。

## 固定テンプレートのページ

以下は固定テンプレートで表示します。

- トップページ: `front-page.php`
- ギャラリー: `template-gallery.php`
- 営業時間＆アクセス: トップページ内の `#home_access`

ギャラリーは `images/auto_gal/` 内の画像を自動で一覧表示します。

## 注意

初期ページ作成はテーマ有効化時に一度だけ実行されます。既に同じスラッグの固定ページがある場合、その本文は上書きしません。
